So this is it, what I've been working on in some form or another for the last several years, and the first of my inventions that I've developed enough that I'm ready to set it loose in the wild. When I started I wasn't even sure if the idea would work, it seemed too simple and straightforward to have not been noticed, so I figured I was probably missing something and it just wouldn't work. The few experiments I managed to hack together seemed to point to the opposite being true however, so I taught myself enough basic electronics to figure it out. From my first real attempt that failed because I didn't properly understand transistors and designed it wrong, to the first working build that required around 26 volts alongside the logic voltage, to the most recent version that crams it all into a single 5 volt source, its been a long process for me. If this is as useful as I think it is though, it'll have been worth every minute. Details in the pdf below.
